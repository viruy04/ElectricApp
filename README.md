# Обучающее приложение ElectricApp

## О проекте

**ElectricApp** — интерактивное обучающее приложение для изучения основных законов постоянного тока с визуализацией электрических цепей. Проект реализован на языке C# с использованием технологии WPF и включает в себя расчетные модули, анимацию движения электронов и теоретические материалы.

## Возможности

### Расчетные модули
- Формулы расчёта силы тока:
- Валидация вводимых данных
- Поддержка различных единиц измерения (А, мА, мкА)

### Визуализация процессов
- Анимированное движение электронов по цепи
- Визуальное отображение зависимости скорости электронов от силы тока
- Графическое представление электрической цепи

### Образовательный контент
- Теоретические материалы по каждой формуле
- Примеры практического применения
- Справочная информация по физическим величинам

## Технологии
- **Язык:** C#
- **Платформа:** WPF (.NET Framework 4.8)
- **Анимация:** Storyboard и DoubleAnimation
- **Стилизация:** XAML Styles и Templates
- **Сохранение данных:** JSON сериализация
- **Тестирование:** Unit testing

## Установка и запуск
**Требования**
- Windows 7 или выше
- .NET Framework 4.8
- 4 ГБ оперативной памяти
- Процессор Intel Core i3 или аналогичный

## Шаги установки
1. Скачайте архив ElectricAppSetup и распакуйте его
2. Запустите .exe от имени администратора, следуйте инструкциям
3. Запустите приложение после установки (Ярлык появится на рабочем столе)
